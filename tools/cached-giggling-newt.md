- 单独UI菜单

| 数字单曲 | 3月 | 1x营收 |
| 实体专辑 | 6月 | 3x营收 |

| 转型 | 继承粉丝 | 加成 |

## 方向一：色情地下产业 + 强奸犯罪系统

### 三大犯罪职业
| 职业 | 门槛 | 收入 | 风险 |
|------|------|------|------|
| 皮条客 | 10万+声望40 | 1.5-8万 | 中 |
| 黑市商人 | 8万 | 2-10万 | 中 |
| 强奸犯 | 无 | 无直接收入 | 极高 |

### 强奸系统
- **双入口**: 街区旅行嵌入选项 + 独立入口 + 熟人暴力
- **抵抗值**: 100→0, 每体位消耗8-15(春药12-20/迷药20-30)
- **堕落值**: 累加8-20/次, >=40下次抵抗减半, >=70不再反抗, >=90斯德哥尔摩
- **怀孕**: 精液>=30(正常40), 秘密出生
- **春药/迷药**: 黑市购买, 交欢前使用

### 各国刑法
| 国家 | 强奸 | 组织卖淫 | 黑市 | 死刑 |
|------|------|---------|------|------|
| 华夏 | 3-15年 | 5-15年 | 1-7年 | 是 |
| 日本 | 5-20年 | 3-10年 | 1-5年 | 否 |
| 韩国 | 5-20年 | 3-10年 | 1-7年 | 否 |
| 新加坡 | 8-20年+鞭 | 5-15年 | 2-10年 | 否 |
| 美国 | 10-25年 | 2-8年 | 1-5年 | 否 |
| 法/英 | 5-20年 | 2-7年 | 1-5年 | 否 |

### 审判
- record<30: 罚款/缓刑
- 30-60: 1-10年
- 60-90: 10-25年
- >90: 无期/死刑(game over, 子女继承50%遗产)

### 坐牢
- 跳过N×12月
- 出狱: 金钱减半, 声望归零, 职业清除, -25%永久求职惩罚

### 皮条客经营
- 购买/租赁店面→招募NPC妓女→月收益=妓女×客单价×客流
- 警察突袭风险, 没收后一切归零

### 黑市商人
- 6种商品(春药/迷药/成人玩具/SM道具/假证/偷拍设备)
- 3段多段叙事交互(简化)

### 警察机制
- 无贿赂, 纯隐匿(藏匿处+反侦察技能)
- 隐匿效率 = hideoutQuality×0.5 + evasionSkill×0.3

### 决策
- 出狱歧视: 全面(-40%录取, -30相亲, -30声望, 禁止贷款), 地下产业不受影响
- 犯罪+心理: 仅数值惩罚, 不联动心理维度
- 偶像被捕: 可洗白(假证+改名+迁居)
- 出狱NPC互动: 引发新类型事件(同学疏远/仇人勒索/受害者联合报案)
- 黑市+街区: 独立入口
- 地下偶像被强迫: 有强迫链(拒绝潜规则后制作人可能用强)

## 方向二：心理/成瘾/创伤系统

### 性瘾(玩家)
```js
state.psychology = { sexAddiction: 0, guilt: 0, corruption: 0 }
```
| 瘾值 | 效果 |
|------|------|
| 30-50 | 心情-1, 智力-2 |
| 50-70 | 心情-3, 绩效-5, 怀疑度+1 |
| 70-85 | 心情-8, 绩效-12, 随机红灯区消费 |
| 85-100 | 心情-15, 绩效-30, 冲动消费, 魅力-10 |

- 增长: 交欢+1~3, 红灯区+3, 约炮+2, 强奸+5
- 戒治所: 5000-15000(按国家), 跳3月, 瘾值-40~60

### NPC被强奸后性瘾
- trauma/3概率触发反向性瘾
- 性瘾>50: 辞职成妓女, 主动勾引玩家/NPC, 无法正常工作

### 罪恶感+堕落(玩家)
- 罪恶感20-50: 与配偶互动文本变化
- 罪恶感50-80: 心情-3, 怀疑度+2
- 罪恶感>80: 自动坦白
- 堕落>50: 正常交欢兴奋度减半
- 堕落>80: 魅力-15

- 降低: 捐款5000(-15), 向配偶坦白(-30但怀疑+50), 寺庙(-3/月)

### NPC创伤+自杀
- 被强奸→trauma+15~30→affection-20, trust-30
- trauma>50: 不再外出
- trauma>80: 概率自杀(pendingDecision)→赶去救(60%)/通知家人(40%)/无视(死亡,record+40)
- trauma每月-1(自然恢复, 玩家不可付费治疗)

### 决策
- 戒治所: 任何时间可选, <50效果减半
- NPC被奸文本: 分来源(玩家/其他NPC/陌生人)
- NPC勾引: 对玩家+NPC都有效
- 偶像性瘾: 有职业影响(scandal+5/月)
- 自杀: pendingDecision+选项
- 斯德哥尔摩: 病态依恋
- 戒治所费用: 华夏5000/日本8000/美国15000/新加坡12000
- 罪恶感: 可主动降低(捐款/坦白/寺庙)
- NPC创伤: 自然恢复(不能付费)
- 性瘾+多角: 性瘾>=70时伴侣嫉妒额外+3/月

## 方向三：NPC主动互动

### 8种主动联系事件
| 事件 | 条件 | 后果 |
|------|------|------|
| 前女友/炮友联系 | 历史, affection>40 | 复合/借钱/怀孕 |
| 金主私信 | 福利姬, 粉丝>3000 | 分粉丝量级文本 |
| 妓女回头客 | 同一NPC>=3次 | 温情向约会 |
| 同学聚会 | 旧同学同城 | 完整事件链(去/不去) |
| 仇人报复 | 离婚/拒绝/被强奸 | 报警/造谣/威胁 |
| 报案 | trauma>30 | record+10~20 |
| 堕落NPC勾引 | NPC性瘾>50 | 主动邀约 |
| 前女友怀孕 | 概率 | 三个选项(承认/否认/逃避) |

### UI
- 右上角浮动📨按钮+红色角标(iOS通知风格)
- 半屏bottom sheet, 依次弹出事件
- 仅中考/高考志愿强制弹窗
- NPC主动性: 三档开关(高/低/关)

### NPC间事件: 仅旁观log
### NPC间谣言: 有传播系统
### NPC勒索: 无

### 决策
- 事件顺序: 时间顺序
- 谣言: 有
- 同学聚会: 完整事件链(花费+跳1月+恢复联系+可能一夜情)
- 金主私信: 分粉丝量级
- 地下偶像粉丝: 有专属互动(更危险)
- 多NPC报案: 独立报案, 不联合
- NPC履历: 记录与玩家互动+独立生活

### 炮友→恋人: 不能转换(两条路互斥)

## 方向五：偶像产业深化

### 偶像团体(系统自动)
- 团名: 随机生成(Sakura7/月光少女/NeoStars等)
- 凝聚力: <30技能减半, <15退团
- 不和: 实力差距/资源不均/人气差距/三角恋

### 年度选举(纯计算)
- votes = fans × (coreRatio×1 + casualRatio×0.5) × charm/50
- 前3: fans×1.5, 收入×2
- 4-10: ×1.1
- 连续2年垫底→降级

### 粉丝类型: core/casual/anti/stalker
### 恋爱禁止: 签约制, fans+15%但暴露惩罚×2
### 黑粉: 伪造丑闻/整容谣言/恐吓信/私生饭跟踪
### 反制: 安保(2000/月)/法律(10000)/发声(免费)
### 海外发展: fans>=10万可去日韩, 适应1-3月
### 偶像→创者: 完全切换, 粉丝继承, 前偶像写入履历
### 毕业公演: 仅自动参加者
### 写真集: 三档尺度(普通1x/性感2x/大尺度3x)
### 退役过渡: 1个月空窗期

## 方向六：角色详情页重构

### 人生履历时间轴
- 追踪: 教育/就业/婚育/职业变换/犯罪/入狱/整容/死亡
- UI: 卡片错开排列(左一个右一个)
- 位置: 页面最后面(附录式)
- 上限: 30条, 超出删除最旧(保留出生和死亡)
- 旧存档: 首次查看时追溯生成
- 已故NPC: 最后一条标灰, 整条灰色调
- 犯罪: 模糊显示("被司法机关拘留")
- 地下偶像: 单独条目

### 详情页布局
- 单栏滚动: 小头像+信息→外貌→遗传→互动→时间轴
- 以NPC视角写(不直接写玩家名字)
- 完全自动记录, 玩家不可手动备注

## 方向七：地下偶像系统

### 入门
- 12岁, 低门槛+降级通道(正规偶像落选→地下)
- 通过求职面板(辍学后)
- 练习时间缓慢提高魅力(+0.5/月)

### 三阶段
- 地下练习生→地下出道→(转正或滑落)

### 月度Livehouse演出
- 收入=粉丝×票价×魅力系数
- 每月1次保底+可选追加(特别公演, 体力-20, 收入×2)
- 3层粉丝量叙事文本

### 潜规则
- 月度概率事件: 制作人/金主/经纪人三种
- 接受→粉丝+收入↑, 可能怀孕
- 拒绝→资源减半, 且制作人可能用强

### 转正(缺一不可)
- fans>=2万 + charm>=55 + 花钱买通 + 潜规则制作人

### 坠落缓冲(3个月)
- 缓冲期可: 借钱/金主包养/疯狂潜规则/最后冲刺
- 失败→福利姬(fans×0.3)或妓女(fans归零)
- 性瘾加速坠落: 缓冲期缩为1月

### 团体
- 同城>=3→自动地下团(2-4人)
- 比正规团更不稳定(凝聚力×2衰减)
- 排名竞争(争夺Livehouse黄金时段+金主资源)

### 终局: 仅两种(转正或滑落, 无平淡退役)
### 被强迫: 不可举报(灰色地带)
