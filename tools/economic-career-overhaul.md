# 方向八：经济系统重构 + 求职UI重构 + 职业晋升轨道

## 概述

重构三个相互关联的系统：经济（税务/银行/公司经营/股票董事）、求职UI（职业专属面板分离）、职业晋升（每个职业线有独立轨道名称和条件）。总新建文件3个，修改文件8+个。

## 第一部分：税务系统

### 各国税率

| 国家 | 单一税率 |
|------|---------|
| 新加坡 | 8% |
| 日本 | 15% |
| 华夏 | 20% |
| 韩国 | 25% |
| 美国 | 30% |
| 法国 | 35% |

### 征收方式

- **年度申报**：每年1月通过待处理事件弹出税务申报
- 事件文本："{国家}税务局通知：你去年的总收入为{金额}，应缴税款{金额}。"
- 三个选项：
  - 如实申报：正常扣税
  - 虚报收入：避税50%但概率(25%)被审计→信用-20+补税+罚款
  - 请会计师：花费=应纳税额×10%，合法降低税负15%

### 避税手段

| 手段 | 效果 | 成本/条件 |
|------|------|---------|
| 迁居低税率国 | 税率降低 | 迁居费+城市适应期 |
| 慈善捐款 | 捐款额×0.3抵税 | 纯粹金钱支出 |
| 产业洗钱 | 犯罪收入自动按产业比例洗白 | 需拥有合法产业（自动，与犯罪系统联动） |

## 第二部分：银行/贷款系统

### 信用分

```
state.finance = {
  creditScore: 50,      // 0-100, 初始50
  loans: [],            // [{id, type, amount, rate, remaining, monthlyPayment}]
}
```

| 信用分 | 效果 |
|--------|------|
| >=70 | 所有贷款利率减半 |
| 50-69 | 正常利率 |
| 30-49 | 利率×1.5 |
| <30 | 银行拒贷（仅可借高利贷） |
| 0 | 仅高利贷+无法IPO+无法申请信用卡 |

**信用变化**：按时还款+3/月，逾期-15/月

### 三种贷款

| 类型 | 抵押 | 上限 | 利率 | 期限 |
|------|------|------|------|------|
| 消费贷 | 无 | 50,000 | 12% (信用好→6%) | 24月 |
| 商业贷 | 产业 | 产业价值×70% | 7% | 60月 |
| 高利贷 | 无 | 100,000 | 35% (固定) | 12月 |

- **即时申请**：点击→输入金额→即时到账/拒绝
- **逾期惩罚**：仅信用惩罚。无暴力催收
- **破产**：不设破产。信用归零+银行拒贷即为最终惩罚

### 低保（失业救济）

- 条件：失业 + 现金<1000
- 金额：500-1500/月（按国家）
- 副作用：声望-5/月，配偶怀疑度+1/月

## 第三部分：公司创建/经营系统

### 创建流程（多步向导）

```
段0「选择行业」:
  餐饮(低门槛低风险,月入500-3000) | 科技(高门槛高回报,月入2000-15000) | 
  零售(中门槛中回报,月入1000-5000) | 娱乐(联动偶像,月入1500-8000) | 
  制造(稳定但慢,月入2000-6000) | 金融(高风险高波动,月入-5000~20000) | 
  成人(联动妓院/黑市,利润×1.3但record缓慢增长,月入3000-12000)

段1「选择性质」:
  个人独资(无限责任: 破产→个人资产受影响, 但税率低5%) | 
  有限责任公司(有限责任: 只赔投资额, 标准税率) | 
  合伙(与NPC合伙: 需找合伙人, 收入共享, 风险共担)

段2「命名+投资」:
  名称: 系统随机生成(如'东京樱花餐饮') + 玩家可点击修改
  投资: 最低投资额取决于行业(餐饮2万,科技10万,零售5万,娱乐8万,制造15万,金融20万,成人5万)

段3「选择策略」:
  保守稳健(低风险低回报) | 激进扩张(高风险高回报) | 差异化精品(中等风险, 品牌溢价)
```

### 经营阶段

- 每月自动产生收入（基于行业+策略+市场周期+员工技能）
- 月度随机事件（待处理事件）：客户投诉/供应商涨价/员工辞职/竞争对手降价
- 可追加投资扩大规模
- 可雇佣1-5名NPC员工（从城市人口池招聘）

### 员工系统

```
employee = {
  id: '', name: '', skill: 0-100, loyalty: 0-100, salary: 0
}
skill: 影响公司收入(+10~30%)
loyalty: 影响离职概率。低忠诚→可能被挖走或偷钱
salary: 员工可能要求加薪（待处理事件）
```

### 市场周期

| 行业 | 周期特征 |
|------|---------|
| 科技 | 繁荣5年→泡沫3年→恢复。波动最大 |
| 餐饮 | 季节波动（夏季+20%，冬季-10%） |
| 零售 | 跟随经济大环境 |
| 娱乐 | 与偶像/地下偶像行业景气联动 |
| 制造 | 最稳定，几乎无波动 |
| 金融 | 牛市/熊市交替，波动极大 |
| 成人 | 反周期（经济衰退时繁荣+25%） |

### 退出方式

| 方式 | 条件 | 收益 |
|------|------|------|
| 出售 | 任何时间 | 动态估值 = 月收入×12~24（倍数取决于行业+年限+市场） |
| IPO上市 | 经营>=5年+月收入>=5000+信用>=60+record<20 | 转为可交易股票，保留30%原始股 |
| 关闭 | 任何时间 | 回收投资额×60% |

## 第四部分：股票董事系统

### 董事权利

| 持股比例 | 权利 |
|---------|------|
| >=5% (董事) | 分红×2 + 每月董事津贴 + 公司内推(录取率+20%) + 内部消息(提前1月预测股价) |
| >=10% | 以上 + 分红×2.5 |
| >=20% (大股东) | 以上 + 可发起收购 + 可任命CEO + 分红×3 |

### 董事会投票（每12月触发一次，待处理事件）

| 投票类型 | 选项 | 影响 |
|---------|------|------|
| 分红决策 | 提高分红/降低分红 | 短期股价 vs 长期增长 |
| CEO人选 | 支持现任/支持挑战者 | 稳定 vs 高风险高回报 |
| 并购提案 | 支持/反对 | 持股比例稀释 vs 市场扩大 |

### 收购竞争对手

- 条件：持有目标公司>=20%股份 或 现金>=目标公司市值×1.3
- 效果：收购后产业收入×1.5 + 市场份额提升
- 风险：被收购公司原董事可能反抗（股价波动）

### 办公室职员面板 Tab

```
[工作] [晋升] [同事] [职业规划]

工作Tab: 绩效仪表盘(当前值/目标值/季度趋势) + 行动按钮(专注/人脉/培训/加班/项目攻坚)

同事Tab: 公司组织架构图 — 你的位置+上司+下属+同级NPC列表
职业规划Tab: 竖向职业时间轴列表
```

### 妓女/福利姬接客面板 Tab

```
[接客] [金主] [健康] [职业规划]

接客Tab: 今日状态(在岗/休息) + 预约列表(客户名/服务类型/价格) + 
         行动(街头拉客/接受预约/拒绝客户)
        客户来源: 同城NPC池随机 — 熟客(>=3次)→可能升级为金主
金主Tab: VIP名单(长期客户, 特殊标记) + 包养关系管理
健康Tab: 健康风险提示 + STD感染状态 + 建议体检
```

### 皮条客经营面板 Tab

```
[经营] [招募] [财务] [职业规划]

经营Tab: 妓女列表(每人: 姓名/年龄/健康/忠诚度/月收入/状态) + 
         客流量趋势 + 警察风险条 + 
         行动(提升安保/扩大会所/贿赂警察按钮已取消)
招募Tab: 从同城NPC池筛选可招募女性 → 提供薪资+提成 → 入职
财务Tab: 收入支出明细(妓女薪资/房租/杂费/净利润)
```

### 黑市商人面板

```
简单面板: 库存数字 × 买卖按钮

## 第六部分：各职业晋升轨道设计

### 办公室职员 — 按行业线

**科技线**：
```
L0 初级工程师 (入职, 绩效>=10) 薪资基准: 1×
L1 高级工程师 (绩效>=45, 经验>=30, 年资>=2年, 晋升概率45%)
L2 资深工程师 (绩效>=55, 经验>=60, 年资>=4年, 晋升概率35%)
L3 首席工程师 (绩效>=65, 经验>=100, 年资>=6年, 晋升概率25%)
L4 技术总监 (绩效>=75, 经验>=150, 年资>=8年, 晋升概率18%)
L5 CTO (绩效>=85, 经验>=220, 年资>=10年, 晋升概率10%)
```

**金融线**：
```
L0 分析师 → L1 高级分析师 → L2 副总裁(VP) → L3 董事(Director) → L4 执行董事(MD) → L5 合伙人
```

**文职/公务员线**：
```
L0 科员 → L1 副主任科员 → L2 主任科员 → L3 副处长 → L4 处长 → L5 局长
```

**医疗线**：
```
L0 住院医师 → L1 主治医师 → L2 副主任医师 → L3 主任医师 → L4 科室主任 → L5 院长
```

**教育线**：
```
L0 助教 → L1 讲师 → L2 副教授 → L3 教授 → L4 系主任 → L5 校长
```

**法律线**：
```
L0 律师助理 → L1 执业律师 → L2 资深律师 → L3 合伙人 → L4 高级合伙人 → L5 首席律师
```

**创意/设计线**：
```
L0 初级设计师 → L1 设计师 → L2 高级设计师 → L3 设计主管 → L4 设计总监 → L5 创意总监
```

**制造/工程线**：
```
L0 技术员 → L1 助理工程师 → L2 工程师 → L3 高级工程师 → L4 总工程师 → L5 技术副总
```

**服务/零售线**：
```
L0 店员 → L1 资深店员 → L2 副店长 → L3 店长 → L4 区域经理 → L5 运营总监
```

**传媒线**：
```
L0 助理编辑 → L1 编辑 → L2 资深编辑 → L3 副主编 → L4 主编 → L5 内容总监
```

### 创作者 — 按粉丝量

```
L0 新人 (粉丝<1000) 收入系数: 0.5×
L1 千粉创作者 (粉丝>=1000) 收入系数: 0.7×
L2 万粉创作者 (粉丝>=10000) 收入系数: 1.0×
L3 十万粉创作者 (粉丝>=100000) 收入系数: 1.5×
L4 百万粉创作者 (粉丝>=1000000) 收入系数: 2.2×
L5 顶流 (粉丝>=5000000) 收入系数: 3.0×
```

晋升条件：纯粉丝数驱动，无年资限制

### 偶像 — 按阶段

```
L0 练习生 (任何偶像的起点)
L1 新人偶像 (出道, 粉丝>=2000)
L2 人气成员 (粉丝>=50000, 总选举排名>=前50%)
L3 ACE (粉丝>=200000, 总选举排名>=前10)
L4 国民偶像 (粉丝>=500000, 总选举排名>=前3)
L5 殿堂级 (连续3年总选举前3 → 永久荣誉, 收入永久翻倍)
```

### 地下偶像 — 简化三级

```
L0 地下练习生 (入门, 粉丝<500)

L2 地下人气 (粉丝>=5000, 潜规则机会增多)

### 妓女 — 无传统晋升，按"身价"分级

```
身价 = f(魅力, 年龄, 经验次数, 健康, 金主数量)

街头流莺(身价<30): 接客收入低, 高风险

高级交际花(身价60-80): 只接VIP, 高收入
头牌名媛(身价>80): 客人为你竞价, 收入极高
```

```
L0 新人福利姬 (粉丝<1000)

L2 频道红人 (粉丝>=20000, 金主私约价格翻倍)

### 皮条客 — 按妓院规模

```
L0 街头中介 (手下0-2人)
L1 小店老板 (手下3-5人, 有固定店面)
L2 会所经营 (手下6-10人)
L3 连锁老板 (手下11-20人, 多店经营)
L4 地下皇帝 (手下>20人, 控制整个红灯区)

### 黑市商人 — 按库存+客户

```
L0 街头小贩 (库存<10件)
L1 固定摊主 (库存10-30件)
L2 黑市中间商 (库存30-100件, 有固定客户群)
L3 地下供应链 (库存>100件, 垄断某类商品)
```
