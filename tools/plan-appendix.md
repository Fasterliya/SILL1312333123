
## 扩展方向八：经济系统重构 + 求职UI重构

### 税务系统

| 决策 | 选择 |
|------|------|
| 税率模式 | 分国家差异化单税率 |
| 征收方式 | **年度申报**(待处理事件, 可选申报方式) |
| 避税手段 | 三种: 迁居到低税率国 + 慈善捐款抵税(捐款×0.3) + 前端产业洗钱(自动) |

**各国税率表**:

| 国家 | 税率 |
|------|------|
| 新加坡 | 8% |
| 日本 | 15% |
| 华夏 | 20% |
| 韩国 | 25% |
| 美国 | 30% |
| 法国 | 35% |

### 银行/贷款系统

| 决策 | 选择 |
|------|------|
| 贷款类型 | 三种: 消费贷(无抵押,上限5万,利率12%) + 商业贷(产业抵押,上限70%产业价值,利率7%) + 高利贷(无门槛上限10万,利率35%) |
| 信用分 | **动态数字分**: 初始50, 按时+3/月, 逾期-15/月。>=70利率减半, <30拒贷, =0仅高利贷 |
| 申请流程 | **即时申请**即时到账 |
| 逾期惩罚 | 仅信用惩罚, 无暴力催收 |
| UI入口 | **资产页内银行子页**: 信用分/贷款列表/申请新贷款 |
| 低保 | 失业+现金<1000→政府发放500-1500/月(按国家)。领取期间声望-5/月 |

### 避税手段

1. **迁居避税**: 迁到新加坡(8%)vs法国(35%)
2. **慈善捐款抵税**: 捐款额×0.3抵扣税款
3. **洗钱(与犯罪联动)**: 拥有产业时犯罪收入自动按比例洗白。产业越多洗白效率越高。在财务摘要显示'本月洗白金额'

### 上市公司股票/董事系统

| 决策 | 选择 |
|------|------|
| 董事权利 | **全面权利**: 分红翻倍+董事津贴+公司内推(录取率+20%)+内部消息(提前1月预测股价) |
| 大股东(>=20%) | 可收购公司+任命CEO+分红×3 |
| 董事会投票 | **三种类型**: 分红决策/CEO人选/并购提案。每12月触发一次 |
| 收购竞争对手 | 可收购: 用持股或现金收购→产业收入×1.5 |

### 公司创建/经营系统

| 决策 | 选择 |
|------|------|
| 创建流程 | **多步创建向导**: 选行业→选性质→命名(系统随机+可改)→投钱→选策略 |
| 行业方向 | **7个**: 餐饮/科技/零售/娱乐/制造/金融/成人 |
| 公司性质 | **有法律后果**: 个人独资(无限责任)/有限责任公司(有限责任)/合伙(与NPC合伙) |
| 经营事件 | **有待处理事件**: 客户投诉/供应商涨价/员工辞职/对手降价 |
| IPO上市 | **严格条件**: 经营>=5年+月收入>=5000+信用分>=60+无犯罪记录(record<20) |
| 出售定价 | **动态估值**: 月收入×12~24(倍数取决于行业+年限+市场) |
| 成人行业联动 | 利润高30%但record缓慢增长。可选合法/地下运营 |
| 员工系统 | **简化管理**: 1-5名员工, 技能(+收入10~30%), 忠诚度(影响离职), 可能被挖走 |
| 产业面板 | **全新经营面板**: 公司列表→点击进详情(收入趋势/员工/事件历史)→操作: 追加投资/出售/上市/关闭 |

### 行业市场周期

**有周期波动**:
- 科技: 繁荣5年→泡沫3年→恢复
- 餐饮: 季节波动(夏高冬低)
- 成人: 反周期(经济衰退时繁荣)

### 职业面板分离

**架构**: 每个职业类型独立专属面板, Tab栏手动切换

| 职业 | Tab | 面板内容 |
|------|-----|---------|
| 办公室职员 | 工作/晋升/同事/职业规划 | 绩效仪表盘+组织架构图+晋升轨道可视化(职员→主管→经理→总监→VP→总经理) |
| 自由职业者 | 工作/职业规划 | 保持现有(发布作品/直播/商务/漫展) |
| 创作者 | 频道/职业规划 | 现有creatorCareer.render |
| 偶像 | 训练/演出/团体/职业规划 | 现有idolSystem.render |
| 地下偶像 | 演出/潜规则/团体/职业规划 | 地下专属面板 |
| 妓女/福利姬 | 接客/金主/健康/职业规划 | 客户预约列表+金主VIP名单+收入统计+健康风险 |
| 皮条客 | 妓院经营/招募/财务/职业规划 | 妓女列表(每人年龄/健康/忠诚/收入)+客流量趋势+警察风险 |
| 黑市商人 | 库存/客户/职业规划 | 简单面板: 库存数字+买卖按钮 |

### 晋升名称分职业线

```
科技线: 初级工程师→高级→资深→首席→技术总监→CTO
金融线: 分析师→高级分析师→VP→董事→MD→合伙人
创作者: 新人→万粉→十万粉→百万粉→顶流
偶像: 练习生→新人→人气成员→ACE→殿堂
```

### 职业规划页面

- **嵌入工作面板**的最后一个Tab
- 竖向列表: 每行一份工作(时间/职位/薪资/离职原因)
- **求职保持单步**: 点击应聘→roll→成功/失败(流程不变)

### 杂项

| 决策 | 选择 |
|------|------|
| 副业 | 不支持。仅主职业有面板。但可同时拥有被动资产(产业/股票/房产) |
| 换工作冷却 | 辞职后2个月竞业禁止(同行业不可应聘)。跨行业无冷却。每满1年可拿1月遣散费 |
| 通货膨胀 | **无**。物价和薪资保持固定 |
